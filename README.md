# Lab 9 - JavaScript Error Handling

Static demo for the CSE 110 Lab 9 assignment.

## Pages URL

https://nguyensd0317.github.io/Lab9_Starter/

## What is included

- Console API demos for `log`, `error`, `table`, `dir`, `dirxml`, `group`, `time/timeEnd`, and `trace`.
- Extra starter-button demos for `count`, `warn`, `assert`, and `clear`.
- Calculator logic using `try`, `catch`, `finally`, and `throw`.
- Custom error classes that extend `Error`.
- Global error handling with `window.onerror` and an `error` event listener.
- Optional TrackJS loading from the token field in the page.

## TrackJS screenshot

1. Create a TrackJS trial account.
2. Copy the TrackJS project token into the page's TrackJS token field.
3. Click **Enable**.
4. Click **Trigger a Global Error** a few times.
5. Capture the TrackJS dashboard showing your username and the reported errors.
6. Save the screenshot in `screenshots/` before submitting.
